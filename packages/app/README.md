# Oky app V2

Set the latest node version as your default

```bash
nvm alias default 20.12.2
```

Allow husky script

```bash
chmod +x ./.husky/_/pre-commit
```

Install packages

```bash
yarn
```

Start

```bash
npx expo start
```

Switch to ExpoGo:
`s`

Start:
`i` to start iOS, `a` to start android

Install a new package

```bash
npx expo install <package_name>
```
