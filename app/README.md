# Oky app V2

Recommended node version: 20.16.0

Allow husky script

```bash
chmod +x ./.husky/_/pre-commit
```

Install packages

```bash
yarn
```

Start

```bash
npx expo start
```

You can also use ngrok:

```bash
npx expo start --tunnel
```

Switch to ExpoGo:
`s`

Start:
`i` to start iOS, `a` to start android

Install a new package

```bash
npx expo install <package_name>
```
